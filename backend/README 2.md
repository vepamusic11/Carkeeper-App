Para poder levantar esto localmente primero deberas clonar el repo en tu pc (asegurate de tener el archiv .env)

Una vez clonado, en la terminal hacer un npm install para instalar todos los paquetes

y luego npm run dev para correrlo. Va a quedar levantado en el puerto 4008.

IMPORTANTE: Asegurate de tener instalado node y npm en tu compu.
